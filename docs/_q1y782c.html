投稿文章

为了更好的发展，本站开放 文章投稿功能，欢迎大家投稿高质量文章~

投稿要求

  • 作品为本人原创，转载他人文章需要标明出处。
  • 不接受带有强烈个人观点的文章。
  • 不接受让人不舒服、用词非主流的文章。
  • 不接受广告类(包括但不限于 SS/VPN/IDC 广告)文章。
  • 不接受违反国家法律法规(反动、色情等)类文章。
  • 不接受200字以下的文章。

注意事项

  • 投稿者默认对投稿文章拥有版权或原作者允许转载，本站不会对其真实性进行校验。
  • 已投稿的文章，说明投稿者授权本站发布该文章。
  • 已在其他站点发布的文章，本站会添加原文链接。
  • 并非所有投稿都会被本站接受，被拒稿件将会被删除，且不会通知投稿者。
  • 所有解释权归 逗比根据地[doub.io]所有。

投稿步骤

  • 认真浏览 投稿要求和注意事项。
  • 使用 TXT文本 或者 Word文件 保存文章内容，图片素材等请用 外链或网盘打包。
  • 通过下面的联系方式把你的 TXT文本/Word文件和图片素材链接（如有）发送给我。

联系方式：

  • 邮箱：[email protected] （邮件标题/主题 请在开头加上 [投稿文章] ，以避免注意不到）
  • Telegram：https://t.me/doubi

投稿后，我会审查投稿作品质量，如果审核通过，我会通过联系方式回复给您，并对投稿文章进行简单合理排版，标注投稿者以发布。

如果有需要，我可以帮助建立一个逗比根据地博客账号，用于投稿和编辑自己投稿的文章，需要在投稿的时候通过联系方式发送你要注册的 用户名(不支持中文)+邮箱+密码 。
