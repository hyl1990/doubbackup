PayPal 更换汇率结算方式 降低手续费，PayPal汇率结算 改为 银行汇率结算

最近一段时间通过 PAYPAL 付款美元，发现要比国际汇率多很多，仔细研究了一下发现，PAYPAL 在前段时间增加了一个新功能，但是却没有通知用户，把用户的汇率结算系统都改成了 PAYPAL 自己的汇率结算系统，结果 PAYPAL 自己的汇率比国际汇率高多了，所以我就写这个教程，教大家如何把结算方式更改为银行汇率结算，不要让 PAYPAL 继续偷偷的坑你钱！

其他PAYPAL教程：PayPal取消自动循环付款/自动续费教程

使用PAYPAL汇率和使用银行汇率的区别是（汇率时间是 2018/04/11）：

  1. PAYPAL汇率：你消费了 10美元 ，PAYPAL根据自己的汇率 6.54XX 计算为 65.4XX 元人民币，然后向你的银行卡的银行申请 65.4XX 元人民币。
  2. 银行结算汇率：你消费了 10美元 ，PAYPAL直接向你银行卡所在银行申请 10美元，然后银行根据自己的汇率 6.28XX 计算需要 62.8XX 元人民币。

这就是两者的差别，PAYPAL自己的汇率更高，这导致了消费10美元会让你多掏2.6元人民币。

管理资金来源 直达连接（评论 @SouiChi 提供）这个链接可省略下面几个步骤，直达 #管理资金来源

账户首页

首先进去你的PayPal账户首页：https://www.paypal.com/businessexp/summary

然后点击右上角的 用户信息 — 用户信息与设置。

我的用户信息

旧版界面

在我的用户信息中，点击 财务信息 — 我的预核准付款 — 更新 。

新版界面

点击 付款 — 管理自动付款 。

我的预核准付款

然后会出现这个页面，点击 设置可用资金来源 。

管理资金来源

这里会列出你的所有资金来源，点击你的 银行卡/信用卡 卡号后面的 兑换选项 。

兑换选项

然后会出现下图，并且默认选择的是：用PayPal的兑换程序完成使用我的卡币种的交易

这个的意思就是，汇率结算按PAYPAL自己的汇率系统来，然后他直接让你的银行 结算人民币。

改成选择第二个选项：在给我的账单中使用卖家列出的币种

这个就是，PAYPAL让银行结算美元，然后银行自己根据汇率兑换人民币。

然后点击 提交 ，就会又回到上一个步骤了，其他的银行卡/信用卡 都重复这个步骤，全部改完了，就点击上个步骤图片里面的 保存 按钮。

这时候你再去付款美元商品，就不会坑爹的走PAYPAL自己的汇率了。
