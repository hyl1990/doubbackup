百度云是越来越坑了，然而即使这样，使用百度云的用户依然很多，而前段时间 百度云加大了破解限速等方法的打击力度，目前这个软件还是比较安全的。

我以前写过五个 百度云 相关的教程文章，不过大多都失效了，还剩下这两个还能用，但是速度不稳定：

  • 『7/24』百度云网盘不限速 命令行下载工具 —— BaiduPCS 使用教程
  • 百度云 利用油猴脚本绕开百度云管家 获取直接下载链接(IDM多线程下载直链)

今天就在来介绍一个破解百度云限速的软件，有兴趣的可以看看。

ProxyEe Down

ProxyEe Down 是使用本地HTTP代理服务器方式嗅探下载请求，支持所有操作系统 和 浏览器(IE9+)，支持多线程分段下载 和 断点下载。

简单的来说，就是安装这个软件后，让浏览器走这个软件建立的本地代理，访问百度网盘后，下载文件会自动转到 ProxyEe Down，然后就可以多线程下载了。

Github 项目：https://github.com/monkeyWie/proxyee-down

下载

首先你需要下载 ProxyEe Down 软件。

下载地址：Github

如果你是 Windows 32位系统，那么下载：proxyee-down-X.XX-windows-x86.zip

如果你是 Windows 64位系统，那么下载：proxyee-down-X.XX-windows-x64.zip

如果你是 其他系统，则需要安装 JAVA 环境，然后下载：proxyee-down-X.XX-jar.zip

下载解压后，得到一个 proxyee-down-X.XX-windows-xXX 文件夹，然后我们进入该文件夹，找到 proxyee-down.exe 并右键 以管理员身份运行。

安装

因为 ProxyEe Down 是建立一个 本地HTTP代理服务器 的方式来下载百度云文件的，所以如果想要访问/下载 HTTPS 链接，还需要安装证书。

当然，新版本的 ProxyEe Down 启动后会检测是否安装证书，如果没有安装会自动安装。

如果你是使用的 火狐浏览器，那么需要手动安装证书才行，安装教程。

打开软件后（首次打开软件可能需要3-10秒左右），右下角会提示软件已启动。这时候就代表安装证书成功。

因为 ProxyEe Down 是建立一个 本地HTTP代理服务器 的方式来下载百度云文件的，所以需要让百度云网盘和下载链接走这个 本地HTTP代理 才行。

有两种方法来配置代理服务器：

一是：使用软件的自带PAC功能（启动软件后自动设置系统代理设置为 PAC）。

二是：使用浏览器扩展来管理代理服务器（Chrome、Firefox 浏览器使用 SwitchyOmega 扩展）。

SwitchyOmega 浏览器扩展设置代理

点击展开 查看更多

一个轻松快捷管理和切换多个代理的浏览器扩展 SwitchyOmega 使用教程

只要是基于 Chromium 内核的浏览器 或 基于 Mozilla 内核的浏览器 都可以安装 SwitchyOmega 扩展来管理代理。

SwitchyOmega 扩展可以去 官方网站 下载。

安装扩展后，你可以直接导入我配置好的备份文件，就可以直接用了，或者你可以学习怎么配置。

导入备份配置文件

导入后点击浏览器右上角的 SwitchyOmega 按钮，选择 proxyee-down 自动切换 ，然后访问百度云网盘看看。

手动配置扩展

点击扩展管理页面左侧的 新建情景模式... ，然后就会弹出窗口让你填写情景模式内容。

如下图所示，情景模式名称：proxyee-down 全局 ，情景模式类型：代理服务器 ，然后点击 创建 按钮。

然后如下图所示：

代理协议：HTTP ，代理服务器：127.0.0.1 ，代理端口：9999

设置后点击左下角的 应用选项 按钮保存。

接着我们还需要配置一个 自动切换模式，点击扩展管理页面左侧的 新建情景模式... ，然后就会弹出窗口让你填写情景模式内容。

如下图所示，情景模式名称：proxyee-down 自动切换 ，情景模式类型：自动切换模式 ，然后点击 创建 按钮。

然后如下图所示，点击 添加条件 按钮，并添加三条百度网盘的规则：

条件类型：域名通配符 ，条件设置：pan.baidu.com ，情景模式：proxyee-down 全局

条件类型：域名通配符 ，条件设置：yum.baidu.com ，情景模式：proxyee-down 全局

条件类型：域名通配符 ，条件设置：*.baidupac.com ，情景模式：proxyee-down 全局

设置后点击左下角的 应用选项 按钮保存。

最后，我们点击浏览器右上角的 SwitchyOmega 按钮，选择 proxyee-down 自动切换 即可。

最后，我们点击浏览器右上角的 SwitchyOmega 按钮，选择 proxyee-down 自动切换 ，然后访问百度云网盘看看。

如果百度云网盘顶部显示的有 proxyee-down，则说明配置成功（如果百度云没有显示HTTPS错误，说明证书安装正常）。

当需要下载百度云文件的时候，只需要 运行软件，然后 选择这个代理情景模式 就行了，不使用的时候可以切换回 系统代理 或 其他情景模式。

使用步骤

目前有两种下载方式，单独下载适合只下载单个文件，如果速度慢的话，可以试试打包下载，具体效果自行测试。

单独下载

点击展开 查看更多

顾名思义，单独下载就是只下载一个文件的时候，在确保 证书安装正常、代理服务器配置正常后，选择你要下载的文件，然后点击 下载按钮。

如果不出意外，ProxyEe Down 就会嗅探接管下载任务，如下图所示。

文件名：可以自定义也可以默认。

文件大小：文件的大小。

支持分段：该任务是否支持分段多线程下载。

分段数：下载的线程数，最多128线程，线程越多不代表下载速度越快。

路径：保存下载文件到本地的位置（首次设置后以后就默认是这个路径了）。

配置完成后点击 创建 按钮，即可开始下载任务。

从下图可以看到下载速度非常快，至少我满意了，如果你这里下载速度也是很慢，那么你需要试试打包下载。

打包下载

点击展开 查看更多

在百度云下载文件夹或者多个文件的时候，百度云服务器会自动把你要下载的多个文件都打包到一个压缩包中方便下载，如果你单独下载速度很慢，那么可以试试打包下载，一般速度会更快一些。

如果你是直接下载多个文件 或者 文件夹，那么就勾选要下载的 文件或文件夹后 正常点击 下载按钮 就行了。

如果你只打算下载一个文件，那么你就需要在本地电脑上新建一个 新建文本文档.txt ，里面随便写几个字符，然后就上传到你要下载的文件所在的百度云文件夹中。

最后如下图所示，勾选刚才上传的 新建文本文档.txt 和 要下载的文件，然后点击 下载按钮。

大于4GB的打包压缩文件解压姿势

如果你打包下载的文件大小大于 4GB，那么很有可能你使用解压软件会提示文件损坏之类的，那么你可以使用该软件自带的解压功能。

在 ProxyEe Down 网页中，点击左侧 工具 选项 - 百度云解压 - 选择要解压的文件 和 解压目录，并点击 确定 按钮即可。
