推荐一个 生成/管理 可自定义地址的 短链接URL网站

推荐给大家一个 生成/管理 短链接URL 网站，可以不注册去生成一个 可自定义地址的 短链接URL，也可以注册后获得更多的功能。放心全部功能完全免费。

简单介绍

这是一个简单的 短链接URL 生成/管理网站，可以实现 长链接缩短，方便好记，有以下特点：

  1. 支持 隐形URL转发
  2. 支持 显性URL转发
  3. 支持 短链接URL 设置密码
  4. 支持 设置合集页面（或者说是 分类/专辑）
  5. 支持 显示统计信息（国家/语言/设备/浏览器 等）

官方网站：https://url.vps.re/

使用方法

直接生成 短链接URL

如果只是简单的生成使用 短链接URL的话，可以直接在 网站首页(url.vps.re) 生成。

第一行 填写要缩短的长链接，如写 https://github.com/shadowsocksr/shadowsocksr/tree/manyuser

第二行(Alias) 填写要设置的别名(只支持字母和数字)，也就是短链接地址，如写 ssrmanyuser

然后点击 Shorten 按钮生成，等几秒你就会获得一个短链接：https://url.vps.re/ssrmanyuser

这样你就可以把这个链接给别人使用了。

链接短好记，并且可以隐藏长链接URL，不过这里生成的 短链接URL 只能使用，不能修改/删除/查看使用统计信息等，如果需要这些功能，需要注册账号。

注册用户以生成/管理 短链接URL

注册账号后，可以使用更多的功能，不过注册账号是要经过审核的。

登陆页面：https://url.vps.re/login

注册页面：https://url.vps.re/register

注册需要填写：用户名(最少6位)、密码和重复密码、邮箱和重复邮箱。

注册后，需要账号审核通过后，才能登陆和使用更多的功能。

登陆进去后，你会出现在仪表盘(Dashboard) 首页，显示大概信息。

左边侧栏分别有：

┌ Dashboard # 仪表盘首页
├ Links # 短链接URL
│     ├ All Links # 查看/管理 全部短链接URL
│     └ Add Link # 添加 短链接URL
├ Bundles # 合集
│     ├ All Bundles # 查看/管理 合集
│     └ Add New # 添加 合集
├ Stats # 统计
│     ├ Location # 位置
│     ├ Referrer # 推荐人
│     ├ Browser # 浏览器
│     ├ Platform # 平台
│     ├ Language # 语言
│     ├ Device # 设备
│     └ Mobile # 移动设备
├ Settings # 设置
│     ├ Settings # 编辑 个人设置
│     ├ My Profile # 编辑 个人信息
│     ├ Change Email # 修改 邮箱
│     └ Change Password # 修改 密码

虽然后台全都是英文，但是实际上没什么操作。

添加 短链接URL也是，第一行 填写要缩短的长链接，第二行(Alias) 填写要设置的别名(只支持字母和数字)，也就是短链接地址。

然后点击 Submit 按钮提交即可。

如果需要更多的设置，可以点击 第二行别名输入框 下面很小的一个按钮 Advanced ，即显示高级功能。

高级功能选项：

Password # 设置 访问密码

Redirection Type # 选择 URL转发类型
           ├ Default # 默认，可以去 Settings - Settings 中设置
           ├ Direct # 隐性 URL转发
           ├ Counter Page 显性 URL转发
Bundle # 选择合集

# 隐性 URL转发：打开短链接URL后，地址栏依然显示 短链接URL，不会显示 源长链接，可以起到隐藏 源长链接的作用。
# 显性 URL转发：打开短链接URL后，地址栏显示 源长链接。

示例：

加密 显性URL转发：https://url.vps.re/ssrpc

不加密 隐形URL转发：https://url.vps.re/ssr

合集演示：https://url.vps.re/bundle/toyo233/1

其他更多的功能，请注册后自行测试！
